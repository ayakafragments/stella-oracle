# 🌟 stella-oracle

星座からシーシャのフレーバーを提案する診断アプリ。

Shisha Bar STELLA（名古屋・栄）の店舗向けに制作した、
誕生日入力またはランダム抽選でその日おすすめのシーシャフレーバーを提案するWebアプリです。

## 機能

- 誕生日 → 太陽星座を自動判定し、Standard / Nicotine-Free フレーバーを提案
- 「星の導きに従う」ボタンでランダム抽選
- 診断結果をXに投稿
- 公式LINEへの予約導線
- Google Mapリンク
- スタッフへの結果提示モーダル

## 使い方

`stella_diagnosis.html` をブラウザで開くだけで動作します。
サーバー不要のシングルHTMLファイルです。
